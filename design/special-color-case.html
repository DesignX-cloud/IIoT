    承色彩角色提到，必須注意品牌主色Primary colors和Semantic Colors看起來過於相似，視情況手動在飽和度(Saturation)、明度(Value)做出差異。例如：

    以haggen為例，scss原檔設定為$primary haggen主色淡化後和紅色過於接近，讓人和danger語意混淆，所以從輔助色中挑選來校正。

    .form-control:focus {
    border-color: $primary;
    box-shadow: 0 0 0 0.2rem rgb($primary/ 25%);
    }
    • 修正前結果

    所以我門從Haggen色票挑選不會和Semantic Colors語義顏色衝突顏色， 這裡挑選$info-25: #c287fa;

    .form-control:focus {
    border-color: $info-25;
    outline: 0;
    box-shadow: 0 0 0 0.2rem rgb($info-25/ 25%);
    }

    編譯後為

    .form-control:focus {
    border-color: #835ca8;
    box-shadow: 0 0 0 0.2rem rgb(131 92 168 / 25%);
    }
